# 10가지 핵심 요소 예측 시스템 완전 통합

## 문제 정의

### 현재 상태
1. **데이터 수집**: app.py에서 부상 정보, 코칭스태프 정보 수집 ✅
2. **예측 호출**: sport_router.py가 모든 파라미터를 받음 ✅
3. **예측 구현**: 각 종목별 predictor가 10가지 요소를 사용 ✅
4. **문제점**: 
   - 선수 개개인의 상세 능력치가 예측에 미반영 ❌
   - 선발/후보 라인업 정보가 수집되지 않음 ❌
   - 실시간 전략/전술 변화 반영 시스템 없음 ❌
   - 승률과 점수가 별도 계산되어 비례하지 않음 ❌

### 목표
모든 리그에서 10가지 요소가 완전히 반영되고, 승률과 점수가 수학적으로 연동되는 시스템 구축

## 요구사항

### 1. 선수 개개인 능력치 반영
- [ ] 선수별 상세 능력치 데이터 구조 정의
- [ ] 능력치를 팀 전력으로 집계하는 함수 구현
- [ ] 각 종목별 predictor에 선수 능력치 반영

### 2. 선발/후보 라인업 시스템
- [ ] 라인업 입력 UI 추가 (app.py)
- [ ] 라인업 데이터 구조 정의
- [ ] 선발 vs 후보 능력치 차이 반영
- [ ] 종목별 포지션 중요도 가중치 적용

### 3. 실시간 전략/전술 변화
- [ ] 전술 옵션 입력 UI (공격형/수비형/균형형)
- [ ] 전술 변화가 예측에 미치는 영향 모델링
- [ ] 상대 전술과의 궁합 계산

### 4. 승률-점수 완벽 연동
- [ ] score_predictor.py 제거 또는 통합
- [ ] 각 predictor에서 점수 분포 → 승률 자동 계산
- [ ] 두 시스템 간 불일치 제거

## 설계

### 데이터 흐름
```
app.py (데이터 수집)
  ↓
  - 팀 기본 데이터
  - 부상 정보 ✅
  - 코칭스태프 ✅
  - 선수 능력치 (추가 필요)
  - 라인업 정보 (추가 필요)
  - 전술 설정 (추가 필요)
  ↓
sport_router.predict_match()
  ↓
종목별 predictor (soccer/basketball/baseball/volleyball)
  ↓
  - 10가지 요소 계산
  - λ (기대 득점) 계산
  - 점수 분포 생성
  - 승률 자동 도출 ✅
  ↓
결과 반환 (승률과 점수가 수학적으로 연동)
```

### 선수 능력치 데이터 구조
```python
player_stats = {
    'player_name': str,
    'position': str,
    'overall_rating': float,  # 0-100
    'form': float,  # 최근 컨디션 0-100
    'fitness': float,  # 체력 0-100
    'is_starter': bool,
    'sport_specific': {
        # 축구: shooting, passing, defending, pace
        # 농구: scoring, rebounding, assists, defense
        # 야구: batting_avg, era, fielding
        # 배구: spike, block, serve, receive
    }
}
```

### 라인업 시스템
```python
lineup = {
    'starters': [player_stats, ...],
    'bench': [player_stats, ...],
    'formation': str,  # "4-3-3", "5-1", etc.
    'tactical_approach': str  # "공격형", "수비형", "균형형"
}
```
